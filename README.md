# TCC – Sistema de Vendas (E-commerce)

Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso técnico em Desenvolvimento de Sistemas pela ETEC.

---

## 🛠️ Tecnologias Utilizadas
- Shopify (plataforma no-code para e-commerce)

---

## 📖 Sobre o Projeto
O projeto consiste no desenvolvimento de um sistema de vendas online (e-commerce), com funcionalidades voltadas para navegação de produtos, visualização de detalhes, carrinho de compras e finalização de pedidos.

---

## 👩‍💻 Minha Participação
- Organização das funcionalidades do sistema  
- Testes e validação da aplicação  
- Desenvolvimento da documentação do projeto  

---

## 🧠 Aprendizados
Durante o desenvolvimento deste projeto, foi possível:
- Compreender o funcionamento de um sistema de vendas  
- Desenvolver pensamento lógico aplicado  
- Aprimorar a organização e documentação de sistemas  
- Trabalhar em equipe no desenvolvimento de um projeto completo
